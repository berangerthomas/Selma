# Systèmes d'Écriture

![Graphique de présentation des familles de systèmes d'écriture](/assets/images/writing_systems_overview.svg)

## Aperçu
Les systèmes d'écriture sont les méthodes structurées que les humains utilisent pour enregistrer le langage visuellement. Ils ont émergé indépendamment dans plusieurs régions, puis se sont répandus et croisés entre les civilisations. Aujourd'hui, trois grandes familles représentent pratiquement toutes les écritures utilisées dans le monde.

## Les Trois Sources Originales
Les écritures du monde remontent à trois inventions indépendantes :
- **Hiéroglyphes Égyptiens** (vers 3200 av. J.-C.) — ancêtre de toute la famille des alphabets sémitiques
- **Cunéiforme Sumérien** (vers 3100 av. J.-C.) — la plus ancienne écriture attestée, un système logosyllabique
- **Écriture Osseuse Oraculaire Chinoise** (vers 1200 av. J.-C.) — ancêtre de toutes les écritures d'Asie de l'Est

## Types de Systèmes d'Écriture

| Type | Description | Exemples |
|---|---|---|
| **Logographique** | Un symbole = un mot ou morphème | Chinois classique, Sumérien |
| **Syllabique** | Un symbole = une syllabe | Hiragana, Katakana |
| **Alphabétique** | Un symbole ≈ un phonème | Latin, Grec, Arabe |
| **Abjad** | Consonnes uniquement (voyelles implicites) | Phénicien, Arabe, Hébreu |
| **Abugida** | Consonne+voyelle par défaut, les signes diacritiques changent la voyelle | Devanagari, Guèze |
