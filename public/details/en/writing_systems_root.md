# Writing Systems

![Overview chart of writing system families](/assets/images/writing_systems_overview.svg)

## Overview
Writing systems are the structured methods humans use to record language visually. They emerged independently in several regions and later spread and cross-pollinated across civilisations. Today, three major families account for virtually all scripts used worldwide.

## The Three Original Sources
The world's scripts trace back to three independent inventions:
- **Egyptian Hieroglyphs** (c. 3200 BCE) — ancestor of the entire Semitic alphabet family
- **Sumerian Cuneiform** (c. 3100 BCE) — the earliest attested writing, a logosyllabic system
- **Chinese Oracle Bone Script** (c. 1200 BCE) — ancestor of all East Asian scripts

## Types of Writing Systems

| Type | Description | Examples |
|---|---|---|
| **Logographic** | One symbol = one word or morpheme | Classical Chinese, Sumerian |
| **Syllabic** | One symbol = one syllable | Hiragana, Katakana |
| **Alphabetic** | One symbol ≈ one phoneme | Latin, Greek, Arabic |
| **Abjad** | Consonants only (vowels implicit) | Phoenician, Arabic, Hebrew |
| **Abugida** | Consonant+default vowel, diacritics change vowel | Devanagari, Ge'ez |
