# Classical Chinese Script

## Overview
Classical Chinese (*wényánwén* 文言文) refers both to the literary language and the writing system that dominated Chinese civilization for over two millennia. Its script evolved from Oracle Bone Script and Bronze Inscriptions into the standardised **Regular Script** (*kǎishū* 楷書) that forms the basis of modern Chinese characters.

## Script Evolution

| Period | Script name | Characteristics |
|---|---|---|
| 1200–1050 BCE | Oracle Bone | Pictographic, sharp |
| 1050–221 BCE | Bronze Script | More standardised |
| 221–206 BCE | Small Seal | First empire standardisation (Qin) |
| 206 BCE–220 CE | Clerical Script | Brush adapted, flatter |
| 3rd c. CE–present | Regular Script | The modern base form |

## Logographic System
Classical Chinese uses a logographic system where each character typically represents one morpheme. The ~50,000 characters in historical dictionaries include:
- **Pictographs** (象形): direct visual representations
- **Ideographs** (指事): abstract concepts
- **Compound ideographs** (會意): meaning from combined elements
- **Phono-semantic compounds** (形聲): ~80% of all characters, combining a meaning component with a phonetic component

## Cultural Influence
Classical Chinese functioned as the *lingua franca* of East Asian intellectual life for 2,000 years. Korean, Japanese and Vietnamese scholars all learned to read and write it — just as European scholars used Latin.
