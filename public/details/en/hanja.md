# Hanja (Korean)

## Overview
**Hanja** (한자, 漢字) are Chinese characters as used in the Korean language. Korea adopted Chinese script around the **2nd–3rd century CE** and used it as the primary writing system for over a millennium before the creation of Hangul in 1443.

## The *Idu* and *Hyangchal* Systems
Before Hangul, Koreans developed clever workarounds to write Korean using Chinese characters:
- **Idu** (吏讀): a system using Chinese characters both logographically and phonetically to represent Korean grammatical elements
- **Hyangchal** (鄕札): an earlier phonetic system used to transcribe Korean poetry

These systems foreshadow the logic that King Sejong would later systematize in Hangul.

## Hanja Today
After Hangul's creation, Hanja use declined progressively:
- **South Korea**: Hanja is taught in secondary school (1,800 characters), occasionally used in newspapers, legal documents and proper name disambiguation
- **North Korea**: Hanja is essentially abolished in everyday use

## Hangul Connection
Hanja is an **influential predecessor to Hangul** — not because Hangul characters are derived from Chinese characters, but because:
1. The *concept* of a character representing a morpheme influenced Hangul's design
2. Hangul was designed to be used *alongside* Hanja
3. Many Sino-Korean words (from Chinese via Hanja) can optionally be written in Hanja even today
